Détails du Projet 1
PayPik

PayPik est un widget de paiement en ligne qui simplifie les transactions via des méthodes comme carte bancaire, PayPal et Token. Il offre une intégration facile sur les sites web et un back-office pour la gestion des transactions.

Technologies utilisées: Java, Spring Boot, Angular, Tailwind CSS, PostgreSQL...

Découvrez le projet en ligne : PayPik Demo

Détails du Projet 2
BackOffice

Le back-office de PayPik permet aux partenaires de gérer les transactions, d'analyser les performances et de résoudre les problèmes. Il offre des fonctionnalités de génération de rapports et de consultation des détails des paiements. La sécurité des données est garantie par des technologies avancées.

Technologies utilisées: Java, Spring Boot, Angular, Tailwind CSS, PostgreSQL...

Découvrez le projet en ligne : BackOffice Demo

Détails du Projet 3
PayPikShop

PayPikShop est un site de test conçu pour intégrer et essayer le widget PayPik. Il permet aux utilisateurs de simuler des transactions en utilisant différentes méthodes de paiement, telles que la carte bancaire, PayPal et Token, dans un environnement sécurisé. Ce site offre ainsi une plateforme pratique pour vérifier l'intégration et le fonctionnement du widget PayPik avant sa mise en production.

Technologies utilisées: HTML, Javascript, Tailwind CSS, Git...

Découvrez le projet en ligne : PayPikShop Demo

Détails du Projet 4
MedCars

Le projet MedCars est une application web développée avec Java Spring MVC et Thymeleaf pour gérer la location de véhicules dans le secteur médical. Les utilisateurs peuvent réserver des voitures et suivre leurs réservations en temps réel. L'application utilise Spring MVC pour la gestion de la logique métier et Thymeleaf pour le rendu des pages dynamiques. Elle met l'accent sur la sécurité des données et l'optimisation de l'expérience utilisateur.

Technologies utilisées: HTML, Javascript, Tailwind CSS, Git...

Détails du Projet 4
Management of travel (PFA)

Ce projet consiste en une application web développée avec Spring Boot pour le backend et ReactJS pour le frontend, avec une interface utilisateur stylisée à l'aide de Bootstrap. L'objectif de l'application est de faciliter la gestion des frais de déplacement pour les entreprises ou les individus.

Technologies utilisées: Spring Boot (Spring Data JPA, Spring Security), ReactJS, PostgreSQL
